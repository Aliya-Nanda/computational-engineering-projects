# Computational Engineering & Algorithmic Problem Solving - ITB

## Description
This repository contains a structured collection of Python scripts developed during the Computational Laboratory sessions at Institut Teknologi Bandung (ITB). The primary focus of these projects is to apply computational thinking, mathematical modeling, and numerical methods to solve complex logical problems and mechanical engineering challenges. The codebase includes assignments extracted from the P01 and P04 programming modules.

## Key Features
*   **Automated Data Validation**: Implements built-in verification workflows to ensure numerical engineering inputs fall within safe operational limits before algorithmic execution.
*   **Iterative Simulations**: Leverages control flow structures (loops and conditionals) to automate repetitive engineering calculations and formula evaluations instantly.
*   **Matrix & Array Manipulation**: Handles sequential data structures to map out the distribution of physical variables and engineering parameters.
*   **Functional Equation Automation**: Modularizes engineering equations using reusable functions to facilitate multi-parameter testing and debugging.

## Tech Stack & Core Concepts
*   **Programming Language**: Python 3.10+
*   **Methodologies**: Algorithmic Thinking, Numerical Analysis, Functional Programming, Clean Code Architecture.
