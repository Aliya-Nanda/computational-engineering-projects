# Computational Engineering & Algorithmic Problem Solving

This repository serves as a dynamic digital portfolio showcasing a collection of technical Python scripts developed during the Computational Laboratory sessions at Institut Teknologi Bandung (ITB). The primary purpose of this project is to implement robust computational methods, algorithmic logic, and numerical analysis to solve structural and mechanical engineering problems.

---

## 📋 Deskripsi

Proyek ini dirancang untuk menjembatani teori teknik mekanika dengan implementasi komputasi praktis. Melalui serangkaian skrip Python yang diekstrak dari penugasan laboratorium terstruktur (`16925300-1-P01 16925300.zip.zip` dan `16925300-4-P04_16925300.zip (1).zip`), proyek ini mendokumentasikan proses pemecahan masalah (problem-solving) dari tingkat logika dasar hingga manipulasi matriks multi-variabel. Fokus utamanya adalah membangun kode yang efisien, tervalidasi, dan terstruktur dengan baik guna mensimulasikan parameter rekayasa teknik secara akurat.

---

## ✨ Fitur Utama

*   **Automated Data Validation**: Sistem verifikasi *input* bawaan untuk memastikan parameter numerik rekayasa berada dalam batas aman sebelum dieksekusi oleh algoritma utama.
*   **Iterative Engineering Simulations**: Implementasi kontrol alur logika (*loops* dan *conditionals*) untuk memproses kalkulasi formula matematika berulang secara instan.
*   **Sequential Data & Matrix Parsing**: Kemampuan memanipulasi struktur data larik (*array*) dan matriks untuk memetakan distribusi variabel fisis (seperti tegangan atau gaya statis).
*   **Functional Equation Automation**: Modularisasi persamaan teknik mesin menggunakan fungsi (*functions*) yang dapat digunakan kembali (*reusable components*) untuk pengujian multi-parameter.
*   **Multi-Variable Parametric Optimization**: Logika optimasi komputasional untuk menyaring, mengurutkan, dan menganalisis kumpulan data lokal berdimensi besar.
