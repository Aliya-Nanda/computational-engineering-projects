# Computational Engineering & Algorithmic Problem Solving

This repository serves as a dynamic digital portfolio showcasing a collection of technical Python scripts developed during the Computational Laboratory sessions at Institut Teknologi Bandung (ITB). The primary purpose of this project is to implement robust computational methods, algorithmic logic, and numerical analysis to solve structural and mechanical engineering problems.

---

## 📋 Description

This project is designed to bridge the gap between mechanical engineering theory and practical computational implementation. Through a series of Python scripts extracted from structured laboratory assignments (`16925300-1-P01 16925300.zip.zip` and `16925300-4-P04_16925300.zip (1).zip`), this project documents the problem-solving process ranging from fundamental logic levels to complex multi-variable matrix manipulations. The primary focus is to build efficient, validated, and well-structured code to accurately simulate engineering parameters.

---

## ✨ Key Features

*   **Automated Data Validation**: A built-in input verification system to ensure that numerical engineering parameters fall within safe operational limits before being executed by the main algorithm.
*   **Iterative Engineering Simulations**: Implementation of logic flow control (loops and conditionals) to instantly process repetitive mathematical formula calculations.
*   **Sequential Data & Matrix Parsing**: The capability to manipulate array and matrix data structures to map out the distribution of physical variables (such as stress or static force analysis).
*   **Functional Equation Automation**: Modularization of mechanical engineering equations using reusable functions (reusable components) for multi-parameter testing workflows.
*   **Multi-Variable Parametric Optimization**: Computational optimization logic designed to filter, sort, and analyze large-scale localized datasets.
